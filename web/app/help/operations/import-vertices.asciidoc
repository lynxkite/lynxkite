### Import vertices

Imports vertices (no edges) from a table.
Each column in the table will be accessible as a vertex attribute.
An extra vertex attribute is generated to hold the internal vertex ID.

====
include::{g}[tag=table-import]

[[id_attr]] Save internal ID as::
The name of the extra vertex attribute that is generated for the internal vertex ID. Set it to
empty string if you don't want the internal id exposed as an attribute.
====
